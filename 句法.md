#### 疑问词不前置

	* 子夏云何

* 2. 介词「于」后的疑问代词不提前

* 1. N · 之 · 定语
	* 蚓无爪牙之利
	* 驾八龙之婉婉兮
* 2. N · 定语 · 者
	* 使吏召诸民当偿者
	* 士卒亡者过半矣 
	* 士志于道而耻恶衣恶食者
	* 求人可使报秦者 
	* 而侯人之心翦翦者
* 3. N · 之 · 定语 · 者
	* 马之千里者
	* 凡民之食于土者
	* 石之铿然有声者
* 4. N · 而 · 定语 · 者
	* 君子而不仁者有矣夫
	* 绪绅而不易其志者

	* 宋人有曹商者
	* 门下有毛遂者
	* 王之臣有其妻子于其友而之楚游者
# 省略句
## 宾语省略
* 「以」「为」「与」可省略，「于」「乎」及其他介词不可省略
* 以
	* 子宽以告
	* 欲以伐齐
	* 请以遗之
	* 愿以闻于官
	* 己弗能有，而以与人
	* 民可以乐成，不可与虑始
	* 政身未敢以许人也
* 为
	* 1. 表被动省略
		* 高伯其为戮乎
		* 若属皆为所虏
	* 2. 表目的省略
		* 公与之宴，为赋
		* 愿为市鞍马，从此替爷征
* 与
	* 竖子不足与谋
	* 不可与言而与之言
	* 秦之所与临者
## 介词省略
* 于
	* 1. 于表时空
		* 犹如运之掌上
		* 将军战河北，臣战河南
		* 败荥阳，伤成皋
	* 2. 于表被动
		* 车骑将军张安世始尝不快上
* 以
	* 死马且买之五百金
	* 高祖购求布千金
	* 虫尽靡，又试之鸡
# 被动句
## 有标志
* 见
	* (受事主语)见+V
		* 今日王其步，见雨
	* (施事主语)见+V：见指代“说话人自己”
		* 母兄见娇
		* 陈卓几见杀状 (陈述董卓几乎杀掉自己的情况)
		* 足下相为观察，还以见海
		* 不要见笑、不要见怪、请您见谅
	* 见+V+N
		* 后有渤海陈斐见授此郡
		* 见疑平生过失
	* 见+V+于+N
		* 吾长见笑於大方之家
	* 见+V+焉：焉=于+N
		* 年四十而见恶焉
	* 见谓 / 见以为：被认为
		* 临难不见谓不勇
* 于
	* 困于酒食
	* 屡憎於人
	* 万尝与庄公战，获乎庄公
* 自
	* 父赏御正卫马匹自王 (赏为见赏)
* 与
	* 秦與天下俱罷 (秦为天下所疲)
* 为
	* 为+V
		* 失礼违命，宜其为禽也
	* 为+V+于+N
		* 胥之父兄为戮於楚
	* 为+V+焉
		* 耻为禽焉
	* 为+N+V
		* 战而不克，为诸侯笑
	* **为+N+之+V**
		* 晋文公为骊姬之谮
		* 为越之擒
	* **为+N+V+之**：之用来复指宾语
		* 为猛虎逐之
		* 僧辄为物拒之
	* **为+N+见+V**
		* 烈士为天下见善矣
	* 为+N+所+V
	* 为+所+V：省略动作的施动者
		* 授不降世，为所执耳
	* 所+V：省略为+N
		* 由所杀蛇白帝子
	* **为+N+之所+V**
		* 为人之所制 
	* **为+N+所见+V**
		* 为时辈所见推许
		* 为先王所见奖饰
	* **为+N+之所见+V**
		* 恐为海内人之所见凡愚
