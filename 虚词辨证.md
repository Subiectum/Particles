# 以
## 介词
* 「以 · 宾」前可能有不、无、其、将、欲、能、必等修饰成分
	* 不以回待人
	* 必无以冕服敛
	* 必以裸享之礼行之
	* 欲以诸侯去之
	* 王其以心疾死乎
	* 岂其以千乘之相易淫乐之朦
	* 讨不以师
	* 作大事不以信
	* 人郢必以庚辰
* 自 · N · 以 · V：表示时空、数量、人物的起点
	* 自日中以争，至于昏
	* 子贝自仞以伐庸
	* 四升为豆，各自其四以登於釜
	* 自郤称以别三传矣

# 于
* 引进动词宾语
	* 凡诸侯有四夷之功，則献上于王，王以警于夷
	* 郑伯由是始恶於王
	* 君盍诛於祝固
* 表动作的原因
	* 郑伯喜於王命而惧其不制於齐也，逃归不盟
	* 赵旃求卿未得，且怒于失楚之致师者
	* 余必使尔罢於奔命以死
* 表动作的结果
	* 为礼卒於无别
	* 不能致训，至于用钺
* 表动作的范围
	* 从君于昏也
	* 为平公筑台，妨於农收
	* 莫敖狃于蒲骚之役
* 表动作的条件
	* 君子作法于凉，其敝犹贪，作法于贪，敝将若之何
* 表动作的内容
	* 训之于民生之不易，祸至之无日
	* 予告汝于难
	* 告尔百姓于朕志
## 于 · V
* 「于」表「往」、「在那(里/时)」、「正在」
	* 唯王于伐楚
	* 唯周公于征伐东夷
	* 君子于役
	* 之子于归
	* 明夷于飞
